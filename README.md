# PDF 旋转工具

一个用于轻松旋转 PDF 页面的现代网页应用。使用 Next.js 和 React 构建。

## 功能特点

- 🔄 支持单页或全部页面旋转
- 🖱️ 拖拽上传文件
- 🔍 缩放预览功能
- ⚡ 快速的客户端处理
- 💾 下载旋转后的 PDF
- ♿ 完全支持无障碍访问

## 技术栈

- **框架:** Next.js 14
- **UI库:** React 18
- **PDF处理:** pdf-lib
- **PDF预览:** react-pdf
- **图标:** 自定义 SVG 组件
- **样式:** CSS Modules

## 开始使用

### 环境要求

- Node.js 18.17 或更高版本
- npm 或 pnpm

### 安装步骤
安装依赖
```
npm install
```

启动开发环境
```
npm run dev
```

构建生产环境
```
npm run build
```

启动生产环境
```
npm run start
```
